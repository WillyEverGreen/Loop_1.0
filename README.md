# AutoSRE Platform

This project implements an Automated Site Reliability Engineering platform.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1.  Clone the repository
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Features

- System Logs Navigation
- Interactive Demo Mode
- Service Health Monitoring
- Incident Management
- Automated Remediation
- Root Cause Analysis
- Database Optimization
- ChatOps
